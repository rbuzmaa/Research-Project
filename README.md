
# Research project

This repository contains the code and notebooks developed as part of the **MPHY0042 Research Project** in the Biomedical Engineering programme.

## Repository Structure

| File | Description |
|------|-------------|
| `colon_cancer_cross_dataset.ipynb` | **Main research notebook** — contains the overall pipeline for the project, including data loading, preprocessing, model training, and evaluation across datasets |
| `colon_cancer_resnet18.ipynb` | Supporting notebook using a ResNet-18 architecture for colon cancer classification |

## Project Overview

This project investigates colon cancer classification using deep learning techniques. The `colon_cancer_cross_dataset` notebook serves as the primary codebase, covering:

- Dataset loading and preprocessing
- Model architecture and training
- Cross-dataset evaluation
- Results and analysis
